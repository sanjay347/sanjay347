## 🔬 Research

**Real-Time NEXRAD Dual-Polarization KDP Estimation** *(NOAA-funded)*

Developed and deployed a novel LP-based C++ algorithm for specific differential phase (KDP) estimation from live NEXRAD Level-II radar data, benchmarked against NWS Level-3 operational products. System runs in production on the OU RPG server processing live storm events.

- **Pipeline**: AWS S3 → Java (NetCDF4/CDM) → VCP gatekeeper → C++17 physics engine → Python visualization → SCP deploy
- **Physics engine**: GLPK LP solver, SNR/QC/ρhv masking, Rayleigh/Non-Rayleigh gate classification, 9-panel matplotlib output
- **Collaborators**: Prof. Guifu Zhang (PI), Jacob Carlin (NOAA/NSSL), Paul McCrone (NWS ROC)
